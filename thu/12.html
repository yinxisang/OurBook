# 机械工程学院/工业工程系

本文于2013年录入/高2012级20&27班罗雪

# 学术氛围

在清华这个大牛棚里面，不管哪个系学术氛围都是很浓厚的。

# 活动氛围

作为唯一一个男女比例达到了1比1的工科院系（我们这一届甚至女生人数反超男生），工业工程系是特别和谐的一个系~我们专业很多地方都要与人打交道，所以大家都很热情很喜欢举办各种活动。

# 专业介绍

工业工程系自建系以来，坚持高起点、国际化和开放式办学方针，围绕人才培养和科学研究开展了卓有成效的探索工作。作为清华园里最年轻的院系之一，在各类人才培养、国际化联合人才培养和交流、开展国际水准科学研究等方面取得了突出成就。在现代化建设过程中，大型复杂工业系统的规划、设计、分析、优化等均需要工业工程理论作为基础，以人为本的社会管理和工业管理也需要以工业工程的基本理念和方法作为指导，工业工程人才在建设高度发达的工业强国和建设高度文明和谐的社会环境的进程中都可以大显身手，大有可为。

传统的机械、电子、计算机等专业是研究机械系统、电子系统、计算机系统的规律，进而进行设计、分析和改进的工科专业，而工业工程是对复杂工业系统(包含一定数量和种类的机器、人、信息、产品等)进行规划、设计、分析、优化和运作的管理与工程相结合的专业。工业工程系现有的29名教师中，绝大多数是从国外一流大学学成回国的优秀人才，全部老师均有国外留学或交流访问的经历，均可以进行外语授课。

本系自成立之初就坚持高起点(向国际一流大学看齐)、国际化(采用国际一流英文教材和举办大量国际知名学者授课活动)和开放式(鼓励学生进行国际交流，同时接收了大量留学生，本系研究生中中外学生之比为1：1)办学方针。本科期间出国交换的学生人数大约占全部人数的30～50%，参与国际课程人数占一半以上，另外还有不少学生因参与教师国际合作项目而出国。

在建系十周年之际，针对我系进行的国际上的教学评估表明，我系本科的教学水平（包括教师和学生等多方面的考量）已经进入了全美前十的水平。这在清华校内也是屈指可数的高水平院系。同时我系还在体育赛事中屡获大奖，在全校最著名的科技赛事“挑战杯”中获得过总分第一名的惊人成绩。

# 培养目标

培养既懂工程技术又掌握管理科学知识的高素质人才。培养的学生具有扎实深厚的数理基础理论，良好的外语和计算机能力以及宽广的工程、企业管理、人文社科等方面的基本知识和技能，具有对复杂生产系统、服务系统进行分析、规划、设计、管理和运作的能力。就业方向：作为中国的顶尖大学，清华大学大多数的本科毕业生继续在本校或者国内其它一流学府或者出国攻读硕士或博士学位。多数硕士毕业生和博士毕业生选择到国内的科研机构，政府部门及企业工作。而针对我系来讲，近三年的数据表明，我系的本科毕业生（一个年级60余人），一般有30人左右选择在清华继续读研，30人左右出国留学深造，小于等于10人选择直接就业。出国方面，近三年来本系学生出国深造的学校主要有：麻省理工学院，斯坦福大学，加州大学-伯克利，哥伦比亚大学等。

工业工程系的毕业生，兼有工程与管理的基础，具备良好的沟通、合作及组织管理能力，深受政府机关、科研院所、国内大中型企业、事业单位、外资企业和合资企业青睐，毕业生就业面广，可在制造业、服务业等众多行业担任工业工程师、制造工程师、系统工程师、质量工程师、物流工程师，在政府、企业、事业等单位成为管理人员或公务员。

# 专业课详细介绍

# 课程名称：机械设计基础（1）（2）（3）

课程内容：

1.正投影的基本理论，常用的二维视图表达方法、用于创意与构思的轴测草图、支持现代设计及系统集成的三维实体模型，计算机绘图初步等。在投影理论的基础上，将二维视图、轴测草图、三维实体模型等典型构思及表达方法有机地结合在一起，将徒手绘图、仪器绘图与计算机绘图结合在一起，以图形表达为核心，形象思维为主线，培养学生工程设计的基本能力。根据不同要求，可选讲曲线、曲面，透视图等内容，为机械、土水等专业的后续课打下牢固基础。

2.本门课程主要介绍机械系统方案设计的过程、设计思想、设计理论与方法，其主要知识点为：机构的运动设计，其中包括机构组成原理，常用机构（连杆机构、凸轮机构、齿轮机构、轮系、间歇运动机构、开式链机构）的类型、特点、功能和运动设计方法，机构的组合方式，组合机构的类型及功能，以及组合机构的设计等；机械的动力设计，主要包括机械的等效动力学模型，机械运转过程中速度波动及其调节方法，刚性转子的平衡设计与平衡试验，平面机构的平衡设计；机械系统方案设计，包括机械产品的设计过程，机械总体方案设计中的设计思想，机械执行系统的功能原理设计、运动规律设计、机构型式设计、系统协调设计、方案评价与决策等。

3.机械设计课程通过对典型零部件设计问题的分析，使学生掌握通用零部件的设计原理、方法和机械设计的一般规律，初步具有机械系统的综合设计能力，树立创新意识，培养机械创新设计能力，培养应用设计资料的能力，了解机械设计理论与方法的最新发展。在机械系统整体分析的基础上，着重分析传动类零件（齿轮传动、蜗杆传动等）支承类零件（轴、滚动轴承）及连接类零件（螺纹连接、轴毂连接等）的工作能力设计及结构设计问题。

笔者看法：

大一上只学习了第一册，即《机械制图》，这门被誉为“清华大一新生最恐惧的课程”真的是很让人劳心。空间思维能力是慢慢培养的，多花一些时间在上面也就可以了。不过需要说明的是，这门课程跟我系未来的学习没有太大关联，只是因为属于机械学院，所以必修这门课程。所以即使没有学得很好，还是要保持良好心态~当然为了学分绩更高，自然是能学多好学多好的啦~

使用教材：

1.刘朝儒等，《机械制图》（第四版）及配套习题集，高等教育出版社。

2.申永胜，《机械原理教程》(第二版)，《机械原理辅导与习题》(第二版)，清华大学出版社。

3.吴宗泽，《机械设计》高等教育出版社吴宗泽，罗圣国，机械设计课程设计手册》（第二版），高等教育出版社吴宗泽、刘莹，《机械设计教程》机械工业出版社。

# 课程名称：微积分、线性代数

课程内容：这就是传说中的高等数学了，所以把这两门课放在一起说吧。顾名思义，微积分就是讲微分和积分，线性代数就是涉及几何、代数方面的知识。再简言之，就是加深难度的数学而已。

笔者看法：嗯，假期没事干的时候还是提前预习下吧，学姐表示没看书很后悔啊。选择我们专业的话，越到后面对数学要求是越高的，所以基础课程一定要好好学啊！

使用教材：

1.《微积分》刘智新、闫浩、章纪民编，每学期两册。还在编中，之后可能会重新结集出版。

2.《线性代数与几何（上）（下）》俞正光、鲁自群、林润亮编，清华大学出版社。

# 课程名称：工业工程概论

课程内容：介绍“工程”是什么，“工业工程”又是什么。由现任系主任和建系元老两人分前半期、后半期上课，考核方式为做presentation，考查小组合作和展示能力。

笔者看法：很棒的课程，在大一下期开设，帮助大家了解我们的专业，提前对自己将来的方向有一个认知。

使用教材：《工业与系统工程概论IntroductionToIndustrialandSystemsEngineering》WayneC.Turner,JoeH.Mize,KennethE.Case,JohnW.Nazemetz编，清华大学出版社（全英文教材）。

其他专业课程：大一其实接触不到什么专业课，大部分是基础科学素养的培养，数学能力的提高，还有凭个人兴趣选择的人文、艺术类专业课程。但是值得一提的是，我们系的所有专业课程（包括运筹学这样的数学课）全部使用国外的英文教材，对英语能力有很大的帮助。英语不太好的学弟学妹不用紧张，其实上了大学慢慢来都来得及的^^况且很多东西都是被逼出来的………………

# 分数线参考

675到680
